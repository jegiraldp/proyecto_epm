Introducción al Sistema

La base de datos se realizó con el fín de agilizar el proceso de búsqueda de información referente a las energías alternas (biomasa, geotermia, hidrógeno, mareomotríz, nuclear y solar) de tal forma que se puedan generar reportes con datos específicos y acordes a la necesidad de cada usuario. Una de las mayores ventajas de la base de datos, además de organizar la información, es la posibilidad de actualizar los datos contenidos en ella.

El usuario tiene dos opciones de entrada al sistema: como administrador o en modo de consulta, lo cual permite tener un mayor control y seguridad de la información contenida en la base de datos. La entrada del usuario a la base datos como administrador o como usuario de consulta se define a la entrada del programa Windows.

Despues de haber entrado a la base de datos mediante un proceso de validación de usuario, como administrador o en forma de consulta, el sistema despliega una pantalla con varias opciones de búsqueda, que le permiten acceder con facilidad a la información requerida.

Para utilizar la base de datos solo se requiere un conocimiento básico de sistemas, como por ejemplo el manejo de una barra de herramientas, navegar en una página o continuar con los enlaces de un tema.

La ayuda permite que el usuario pueda resolver las inquietudes que tenga respecto al manejo de la base de datos, incluyendo las opciones con las que cuenta el programa (generación de reportes, impresión, modificación de datos, etc.).
