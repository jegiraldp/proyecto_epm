Glosario del Sistema

 

FUENTE: recurso del cual proviene la energía alternativa. (sol, mar, calor de la tierra, hidrógeno, biomasa).

PROCESO: aplicación científica que posibilita generar energía eléctrica a partir de una fuente.

FORMA DE APROVECHAMIENTO: tecnología que aprovecha la posibilidad de generar energía a partir de un proceso.

VARIANTE TECNOLÓGICA: distintos dispositivos tecnológicos que implementan la forma de aprovechamiento.

RECURSO: materia prima que se usa para el funcionamiento de las distintas variantes tecnológicas, también puede estar asociado con las formas de aprovechamiento.

VARIABLES: son las distintas características ligadas a las fuentes, procesos, formas de aprovechamiento, variantes tecnológicas y recursos.

ARCHIVOS: se denominan así a todas las tablas y figuras que acompañan el informe. El sistema permite el ingreso a estos archivos mediante un enlace generado en la base de datos.
