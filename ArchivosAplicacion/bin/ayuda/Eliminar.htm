Eliminar Información

El sistema cuenta con la posibilidad de eliminar información de la base de datos por medio de operaciones sencillas que serán descritas a continuación. La orden de eliminar algún dato se encuentra en la barra de herramientas, ubicada en la parte superior de la pantalla.

Al igual que para Agregar información, la opción Eliminar puede ser utilizada en:

  • Fuente
  • Proceso
  • Forma de aprovechamiento
  • Variante tecnológica
  • Grupo
  • Variable característica
  • Recurso
  • Usuario

Para proceder con la eliminación de la información, se debe escoger el grupo al cual pertenece, es decir, si la información que se desea eliminar es una Fuente o Proceso o Forma de Aprovechamiento o cualquiera de las opciones anteriormente observadas.

Una vez seleccionada la opción que se desea eliminar, el sistema desplegará un recuadro en el que, por medio de una caja de chequeo, se puede escoger la Fuente, Proceso o la opción que se haya elegido para eliminar.

Después de haber escogido la opción que se desea eliminar, el usuario puede aceptar o cancelar la operación de Eliminar mediante dos botones de comando. Si el usuario decide Eliminar, el sistema despliega una ventana de seguridad que solicita la confirmación del proceso de eliminación.

Eliminar Datos Variables

Si lo que se quiere es eliminar un Dato Variable el proceso es igualente sencillo. Se selecciona el dato a eliminar y se da click en el botón "Borrar dato variable", ubicado en la barra de herramientas de los Datos Variables.

Al dar click en el botón "Borrar dato variable", el sistema despliega una ventana en la que se solicita la confirmación del proceso. En este momento el usuario tiene la posibilidad de Aceptar o Cancelar la operación.

Eliminar Archivos

Para eliminar un archivo se procede similarmente como se hizo con Datos Variables. Primero se debe elegir el archivo que se desea eliminar y después se da click sobre el botón "Borrar archivo adjunto"

Al dar click en el botón "Borrar archivo adjunto", el sistema despliega una ventana en la que se solicita la confirmación del proceso. En este momento el usuario tiene la posibilidad de Aceptar o Cancelar la operación.
